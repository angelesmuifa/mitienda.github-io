 README PRE-PROYECTO
Realizado por: Anges Lucero Muifa
Comence este Pre-proyecto configurando la estructura basica de HTML5.
Dentro de la etiqueta <head> se definio el titulo de la pagina y se vincularon las hojas de estilos externas, ademas de integrar fuentes de Google Fonts e iconos de Font Awesome.
La estructura semantica del <body> se divide en tres partes principales:
Header, Main, footer y Header
header 
Se implementaron tres secciones principales:
-Barra superior
-Contiene iconos de usuario y carrito.
-Incluye el logo de la tienda.
-Se agrego un buscador de productos.
-Menu de navegacion
-Permite desplazarse por las diferentes secciones de la pagina mediante enlaces internos.
-Banner principal
-Incluye el nombre de la tienda y una imagen de fondo decorativa.
Main
Dentro del main se desarrollaron cuatro secciones principales:
Productos
Cree una seccion de productos con tarjetas diseñadas utilizando Flexbox para organizar los articulos de manera responsiva.
Cada tarjeta incluye:
Imagen del producto, nombre, tipo de prenda, precio, selector de colores, boton de carrito de compras y etiqueta de descuento
Ademas, se utilizo un id en el titulo de la seccion para permitir la navegacion interna desde el menu.

Sobre Nosotros

Contacto
